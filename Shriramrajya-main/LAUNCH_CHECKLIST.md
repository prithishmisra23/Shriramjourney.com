## Common Issues & Solutions

### "Sitemap not submitting"
**Solution:**
1. Verify file is at: `/public/sitemap.xml`
2. Check format with [XML Validator](https://www.w3schools.com/xml/xml_validator.asp)
3. Re-submit in Google Search Console

### "AdSense code not firing"
**Solution:**
1. Check `index.html` has AdSense script
2. Verify client ID is correct in `ads.txt`
3. Wait 24-48 hours for Google to verify
4. Check browser console for errors

### "Page not indexed"
**Solution:**
1. Verify sitemap submitted in GSC
2. Check robots.txt allows crawling
3. Ensure page is linked internally
4. Check for crawl errors in GSC
5. Manually request indexing in GSC

### "Low Core Web Vitals score"
**Solution:**
1. Run Lighthouse audit
2. Compress images (use TinyPNG)
3. Enable lazy loading for images
4. Minify CSS/JavaScript
5. Optimize fonts loading
6. See `docs/DEPLOYMENT.md` for details
